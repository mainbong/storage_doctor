# Log Analysis Skill

이 스킬은 로그 파일을 효과적으로 분석하고 문제를 찾는 방법을 제공합니다.

## 주요 기능

### 1. 실시간 모니터링
- tail -f 스타일 모니터링
- 키워드 필터링
- 에러 패턴 감지

### 2. 로그 검색
- 정규식 패턴 검색
- 시간 범위 필터링
- 로그 레벨 필터링

### 3. 로그 분석
- 통계 생성
- 에러 요약
- 패턴 분석

## 사용 방법

1. 로그 파일 경로 확인
2. 적절한 액션 선택 (tail/search/filter/summarize)
3. 필요한 경우 패턴 지정
4. 결과 분석 및 문제 파악
